This guide covers the configuration and management of DDoS protection on F5 Distributed Cloud (XC).

## Prerequisites

- Access to an F5 Distributed Cloud tenant
- Appropriate permissions to configure DDoS protection settings

## Overview

F5 Distributed Cloud provides DDoS protection capabilities that can be configured to safeguard your applications and infrastructure from volumetric, protocol, and application-layer attacks.
